# homebrew-bragfile

Homebrew tap for [bragfile](https://github.com/jysf/bragfile000) — a
local-first Go CLI for capturing career-worthy work moments.

The `bragfile.rb` formula will be auto-published here by
[goreleaser](https://goreleaser.com/) when bragfile cuts its first
tagged release. For source code, install instructions, and issues, see
the main repo: <https://github.com/jysf/bragfile000>.
